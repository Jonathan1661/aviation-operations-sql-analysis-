This project analyzes daily flight volume trends by airport.

## Data
Synthetic dataset of daily flights per airport.

## Queries
- Identify multi-day increasing trends
- Detect day-over-day declines
- Calculate daily flight changes

## Business Use Case

Airline operations teams need to monitor changes in daily flight volume to identify surges, disruptions, or recovery patterns.

This analysis demonstrates how SQL can be used to:
- Detect increasing operational trends
- Identify declines in flight activity
- Measure day-over-day changes in volume
